# Tourism-Reports-LLM

## La Voz del Turista: Generación de Reportes de Inteligencia Turística para los Pueblos Mágicos mediante Modelos de Lenguaje a Gran Escala

### The Voice of the Tourist: Generation of Tourism Intelligence Reports for Magical Towns using Large Language Models

This repository contains the implementation of a master's thesis project focused on generating tourism intelligence reports for México's Pueblos Mágicos (Magical Towns) using Large Language Models (LLMs) and a Retrieval-Augmented Generation (RAG) system.

## Project Overview

The project aims to synthesize comprehensive tourism intelligence reports by analyzing TripAdvisor reviews from visitors to Pueblos Mágicos in México. By leveraging state-of-the-art Large Language Models and RAG architecture, this system transforms scattered tourist reviews into actionable tourism insights.

### Key Features

- **Data Collection**: Retrieval of TripAdvisor reviews from Pueblos Mágicos
- **RAG System**: Intelligent retrieval and synthesis of relevant review data
- **Report Generation**: Automated creation of comprehensive tourism intelligence reports
- **Analysis**: Sentiment analysis, theme extraction, and trend identification
- **Insights**: Actionable intelligence for tourism stakeholders

## What are Pueblos Mágicos?

Pueblos Mágicos (Magical Towns) is a tourism program by the Mexican government that recognizes towns throughout México that offer visitors a "magical" experience due to their natural beauty, cultural richness, historical significance, or unique characteristics.

## System Architecture

The system is built on a RAG (Retrieval-Augmented Generation) architecture that:

1. **Retrieves** relevant TripAdvisor reviews from Pueblos Mágicos
2. **Processes** and indexes the review data for efficient retrieval
3. **Augments** LLM prompts with relevant retrieved information
4. **Generates** comprehensive tourism intelligence reports

## Technology Stack

- **Large Language Models**: For natural language understanding and report generation
- **Vector Database**: For efficient similarity search and retrieval
- **Data Processing**: Python-based pipeline for review collection and preprocessing
- **RAG Framework**: Integration of retrieval and generation components
